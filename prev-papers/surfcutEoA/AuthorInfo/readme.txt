This package contains the LaTeX2e support for
contributions to Springer encyclopedias

The following files are part of the macro package

  readme.txt    This file
  history.txt   The version history of the package
  svcyclop.cls  The document class file for contributions
                to Springer encyclopedias

  template.tex  A fill-in-form for a standard paper with usage comments
  spbasic.bst   BibTeX style file
  figure.eps    a sample image to demonstrate figure inclusion
  figure.pdf    same image in PDF form

  example.tex   Example of an article
  example.pdf   The file example.pdf contains the corresponding output.
  see the corresponding article in the encyclopedia on SpringerLink:
  http://link.springer.com/referenceworkentry/10.1007/978-0-387-30162-4_69

Bibliographic references in author year form are preferably made with
the "natbib" package of Patrick W. Daly. You can find it on CTAN,
e.g. at ftp://ftp.dante.de/tex-archive/macros/latex/contrib/natbib/


INSTALLATION

Copy svcyclop.cls to a place where LaTeX can find them.


GETTING STARTED

You should proceed by merely filling out a copy of the template and then
looking up additional commands as needed.
